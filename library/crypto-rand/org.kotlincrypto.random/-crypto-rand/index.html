CryptoRand

abstract class CryptoRand(source)

A source for obtaining random data from the system, suitable for use in cryptographic operations and/or seeding random number generators (RNGs).

e.g. (Seeding a Random Number Generator)

val rng = Random(
    seed = CryptoRand.Default.nextBytes(ByteArray(Long.SIZE_BYTES))
        // Using KotlinCrypto/bitops endian library
        // to convert 8 bytes to a Long
        .beLongAt(0)
)
